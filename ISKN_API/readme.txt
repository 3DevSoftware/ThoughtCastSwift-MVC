1. Directory structure overview
==========================================================================

  You will find some directories after decompressing the archive in which
  came the API. These are:

 	\ISKN_API    the compiled libraries and libs to link with your programs:
	\ISKN_API\arm_iOS
	\ISKN_API\x64_MACOS_Clang
	\ISKN_API\x64_Linux_GCC
	\ISKN_API\x64_MSVC
	\ISKN_API\x86_MSVC
	\ISKN_API\x86_MingW

  In each sub-folder you also find :\include Header files to include when the ISKN API.

  \examples    Examples showing how to use the ISKN API.
  \lib         Lib to link with your programs when using the engine.

==========================================================================
2. How to start
==========================================================================

  To start developing your own applications with the ISKN API take a look at the ISKN API  presentation and tutorials on our website http://developer.iskn.co/docs.
  You can find all the examples in the \examples directory.
 
  Windows Users: 
  There are Visual Studio as well as Qt projects for the examples.

  Linux users: 
  There are Qt projects for the examples.
  
  Mac users: 
  There is Xcode projects for each example available in iOS and in OS X
